# IT Blog - MVP Monorepository

Це повноцінний новинний блог про інформаційні технології, що є основою для серії лабораторних робіт з курсу **"SEO та пошукова оптимізація"**.

## Структура проєкту

Проєкт організовано у вигляді монорепозиторію:

*   **`apps/frontend`**: Фронтенд-частина на базі **Next.js** (App Router, JavaScript, Tailwind CSS). Обов'язково підтримує Server-Side Rendering (SSR).
*   **`apps/backend`**: Бекенд-сервіс на базі **Node.js** та **Express**.
*   **`project-spec.md`**: Технічне завдання на проєкт "IT Blog".
*   **`lab-01.md`**: Інструкції та контрольні питання до Лабораторної роботи №1.

---

## Швидкий старт

Для роботи з кожним сервісом перейдіть у відповідну папку:

### Бекенд (apps/backend)

1.  Перейдіть до директорії бекенду:
    ```bash
    cd apps/backend
    ```
2.  Встановіть залежності:
    ```bash
    npm install
    ```
3.  Запустіть сервер:
    *   Режим розробки (з автоперезапуском):
        ```bash
        npm run dev
        ```
    *   Звичайний запуск:
        ```bash
        npm start
        ```
    Сервер буде доступний за адресою [http://localhost:5000](http://localhost:5000).

### Фронтенд (apps/frontend)

1.  Перейдіть до директорії фронтенду:
    ```bash
    cd apps/frontend
    ```
2.  Встановіть залежності:
    ```bash
    npm install
    ```
3.  Запустіть Next.js у режимі розробки:
    ```bash
    npm run dev
    ```
    Клієнтська частина буде доступна за адресою [http://localhost:3000](http://localhost:3000).
